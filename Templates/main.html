Commercial Aviation Fatalities Predictor

Most flight-related fatalities stem from a loss of “airplane state awareness”. That is, ineffective attention management on the part of pilots who may be distracted, sleepy or in other dangerous cognitive states.
We have built a model to detect troubling events from aircrew’s physiological data. Using this, pilots could be alerted when they enter a troubling state, preventing accidents and saving lives.

We can process the data in two modes:

1. Single Mode - Process single record. We need to specify the input directly.

2. Batch Mode - Process multiple records. We need to upload a csv file with inputs.
